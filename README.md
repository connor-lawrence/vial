# Hello, World!
This is where I keep the files for my hobby OS development project, `VialOS`, which runs on my hobby kernel named `CorK`!

This the GitHub repo just contains all of the project folder, so expect a mess- because that's what you're going to get anyway!

Feel free to poke around, try things out, and have fun! :)

## Why on Earth are they named `VialOS` and `CorK`?
When I was originally creating my OS project, it's name was `TLOS` (Tiny Little Operating System). I then realized I needed a name for the kernel... since it was the core of the system, I named it CoreKernel, shortened to CoreK, then to `CorK`! I then wanted my whole project to have a uniform naming scheme scheme, so I changed my OS's name to `VialOS`... because... there's a `CorK` in the `Vial`... get it...? (Just let me believe that I'm funny)
